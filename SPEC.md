# ペット飼育費用シミュレーター | 犬・猫の年間コストを無料試算 | AppADayCreator - 技術仕様書

## 概要

**サービス名**: Pet Care Cost Simulator
**バージョン**: 1.3.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/pet-care-cost-simulator/

犬・猫・小動物の種類・体格・医療費を考慮したペット飼育費用を無料シミュレーション。初期費用から年間維持費まで、ペットを迎える前に知っておきたい費用の目安を提示します。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する

## v1.1.0 更新内容（2026-06-03）

### P1改善（品質向上）
- **医療費の種別別補正**: 大型犬は医療費 +30%、中型犬 +15% を自動計算
- **住居情報の費用反映**: マンション住まいで月1,000円追加、引越し予定なら初期費用+10万円
- **医療費の上限設定**: 年間医療費が15万円を超えないように制限（現実的な金額）

### P2改善（ユーザー体験）
- **年間維持費の内訳表示**: フード・医療費・トリミング・保険・その他の費目別を視覚化（棒グラフ）
- **詳細な寿命計算**: ペット種別・サイズで正確な平均寿命を反映
  - 小型犬: 15年、中型犬: 14年、大型犬: 11年
  - 室内猫: 16年、室内外猫: 14年、小動物: 5年
- **保険シミュレーション機能**: 保険あり/なしの経済比較を明示
  - 保険なしの場合：緊急資金の推奨額を表示
  - 保険ありの場合：月額負担と生涯保険料を表示

## v1.2.0 更新内容（2026-06-06）

### P1改善（UX・機能）
- **ラジオボタン自動進行（Auto-advance）**: 選択肢を選ぶと300ms後に自動で次の質問へ進む。最終質問は手動確認（「診断する」ボタン）

- **テスティモニアルをサービス固有の内容に修正**: ペット飼育費用シミュレーターの利用体験を反映した3件の声に修正

## v1.3.0 更新内容（2026-06-10）

### P1改善（データ永続化・ローカルストレージ）【M3実装】
- **入力値の自動保存**: 各質問の選択値を localStorage に自動保存
- **ページ復旧後の自動復元**: ブラウザで同じページを開いた場合、前回の入力値を自動的に復元
- **診断履歴の記録**: 過去5件の診断結果（ペット種別・年間維持費・生涯費用・実行日時）を局所メモリに保存

### P1改善（グラフ・可視化強化）【M4拡張】
- **円グラフの追加（Chart.js統合）**: 年間維持費の費目別割合を doughnut チャートで表示
- **レスポンシブ対応**: PC では 2列レイアウト（棒グラフ + 円グラフ）、モバイルは 1列で自動調整
- **Chart.js CDN 統合**: 軽量な doughnut チャートで結果の可視化を強化（外部API不使用）

### P1改善（エクスポート・印刷機能）【M10実装】
- **CSV ダウンロード機能**: 診断結果をCSV形式でダウンロード（ペット種別・初期費用・年間維持費・寿命・生涯費用）
- **ブラウザ印刷対応**: 「印刷する」ボタンで window.print() を実行。既存 CSS print スタイルでアフィリエイト要素を非表示化

### UX改善
- **シェア・エクスポート UI の拡張**: 従来の LINE・X シェアに加え、CSV・印刷ボタンをグリッドレイアウトで配置
- **結果画面への複合ビジュアル**: 棒グラフ（内訳詳細）+ 円グラフ（割合視認性）で数値理解を向上
